# 打包说明

## 打包为Windows可执行文件

### 前置要求

1. 确保已安装 Node.js 16+ 和 npm
2. 已安装所有依赖：`npm install`

### 打包步骤

1. **执行打包命令**：
   ```bash
   npm run electron:build
   ```

2. **打包输出**：
   - 打包完成后，可执行文件将生成在 `release` 目录中
   - 会生成两种格式：
     - **安装程序**：`广理成绩核查 Setup 1.1.0.exe` (NSIS安装程序)
     - **便携版**：`广理成绩核查-1.1.0-portable.exe` (无需安装，直接运行)

### 打包文件说明

#### 安装程序版本 (NSIS)
- 文件名：`广理成绩核查 Setup 1.1.0.exe`
- 特点：
  - 需要安装到系统
  - 可创建桌面快捷方式和开始菜单快捷方式
  - 可通过控制面板卸载
  - 适合需要正式安装的场景

#### 便携版 (Portable)
- 文件名：`广理成绩核查-1.1.0-portable.exe`
- 特点：
  - 无需安装，双击即可运行
  - 所有文件都在一个exe文件中
  - 适合临时使用或不想安装的场景

### 分发说明

如果需要分发程序，建议：

1. **单独分发**：直接分发 `广理成绩核查 Setup 1.1.0.exe` 或便携版
2. **压缩包分发**：将可执行文件打包为zip压缩包，便于传输
   - 安装程序：`广理成绩核查-安装程序.zip`
   - 便携版：`广理成绩核查-便携版.zip`

### 注意事项

- 首次打包可能需要下载 Electron 二进制文件，请确保网络连接正常
- 打包过程可能需要几分钟时间，请耐心等待
- 如果遇到权限问题，请以管理员身份运行命令提示符
